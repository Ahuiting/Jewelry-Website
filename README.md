# Mirromiro Jewelry Website

A modern, responsive e-commerce website for Mirromiro Jewelry, featuring a clean design and seamless shopping experience.

## Technologies Used
- HTML
- CSS
- JavaScript 

## Project Structure

```
Jewelry-Website/
├── html/
│   ├── index.html
│   ├── shop.html
│   ├── about.html
│   ├── contact.html
│   └── cart.html
├── js/
│   ├── script.js
│   ├── shop.js
│   └── cart.js
├── product_images/
│   └── [product images]
├── style.css
└── products.json
```

## Getting Started

1. Clone the repository:
```bash
git clone [repository-url]
```
2. Open the project in your preferred code editor

3. Open `html/index.html` in your web browser to view the website
